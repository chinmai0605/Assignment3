# Assignment 3 - Functions & Modules in Python

## Task 1: Calculate Factorial Using a Function
This Python program:
- Defines a function `factorial(n)` that calculates the factorial of a number using a loop
- Takes a number as input from the user
- Calls the `factorial` function and prints the factorial in the format: `Factorial of 5 is: 120`

## Task 2: Using the Math Module for Calculations
This Python program:
- Asks the user to enter a number as input
- Uses the `math` module to calculate:
  - Square root of the number
  - Natural logarithm (log base e) of the number
  - Sine of the number (in radians)
- Displays all the calculated results to the user
